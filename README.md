# DoubleTime

A simple scheduler 

Visit the [demoo](https://lit-garden-97808.herokuapp.com/)

### Dependancies
1. psql
2. ruby-2.6.1
3. node 8

### Set up locally

1. clone project
2. bundle install
3. rake db:init
4. rails db:migrate && rails db:seed
5. cd client && npm i
6. cd .. && ... start developing (either "rake start", or "npm run start:dev")
